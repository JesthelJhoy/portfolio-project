What I've built

Vero Weather App

VeroWeather is a responsive weather forecasting web app I built entirely from scratch using HTML, CSS, and JavaScript. The word “Vero” means "real" in Italian—reflecting the app's goal to provide real, up-to-date weather information. Users can search for any city and instantly view a multi-day forecast, complete with temperature, weather conditions, and matching icons. The app fetches real-time data through an API and presents it in a clean, user-friendly interface. This project highlights my ability to work with external APIs, manage dynamic user input, design responsive layouts, and create interactive, real-world frontend applications.

World Clock

The World Clock website is an interactive website that displays the current time in a selection of major cities across different timezones. I developed this project using HTML, CSS, and JavaScript, integrating an external API to retrieve accurate, real-time data for each location. The app features a simple, user-friendly interface that highlights the time differences between various parts of the world. While users cannot search for cities manually, the app includes a curated list of locations to demonstrate the functionality. This project showcases my ability to work with APIs, manage asynchronous JavaScript, and create dynamic frontend applications using core web technologies.

Upliftify App

Upliftify is a mood-based web application designed to provide users with a personalized uplifting message generated by AI. Built using HTML, CSS, and JavaScript, the app allows users to select their current mood, and in response, it fetches a unique, positive message through an AI-powered API. This project highlights my ability to work with modern web technologies and demonstrates how frontend development can be combined with artificial intelligence to create engaging and emotionally supportive user experiences. The clean interface and simple interaction flow make Upliftify a thoughtful and encouraging tool, reflecting both technical skill and user-centered design.
